## ğŸ› ï¸ Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/catppuccin-mocha-dots.git ~/.config
   ```

2. Install required packages:
   ```bash
   paru -S hyprland rofi dunst waybar-cava eww alacritty fish
   ```

3. Copy the configuration files to your system:
   ```bash
   cp -r .config/* ~/.config/
   ```

4. Reboot or log out and log back in to apply changes.
